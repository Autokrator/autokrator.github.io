Game Development

Clash Wars  

Clash Wars is an online multiplayer game where two players face each other in a tapping battle. The goal is to push your energy beam on the other player.

The client side of the game is an android app written in Java using the wrapper framework libGDX. The server is implemented in C++11 and uses a concurrent architecture. A detailed breakdown of the server's threaded design can be found here.

Plants Vs. Zombies Clone  

This was a Plants Vs. Zombies implementation in C++ using the Qt framework. Making this game taught me a lot regarding object-oriented programming. The implementation follows core OOP priciples and utilizes Qt's graphics libraries for visuals and IO libraries to save player data.




Web Development

Quick Links    

Quick Links is a Chrome extension written in HTML, CSS and JS that allows you to assign hotkeys to desired links. It utilizes the Chrome API to to achieve this. It is designed to give a modern feel supporting a dark and light theme.



LeagueBets (In Developemnt)

LeagueBets is currently in development, but it is one of my most ambitous projects. It utilizes a vareity of things I have learned this year in web development.

LeagueBets is a single-page MEAN stack web application that allows registered users to place bets on active League of Legends matches using virtual tokens. The users can register for the site using Facebook, Google+ or local site registration. This is achieved by using the Riot API which allows the retrival of League of Legends feautered game data along with Facebook and Google APIs for OAuth 2.0 authetication.
